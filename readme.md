# Tour River Plate - Museo y Estadio Monumental

## Descripción del Proyecto
Este sitio web es una plataforma informativa y de reserva para el Tour oficial del Club Atlético River Plate. Permite a los usuarios conocer las diferentes opciones de pases disponibles para visitar el Museo River y el Estadio Monumental, leer testimonios de otros visitantes y enviar consultas de reservas grupales.

## Requisitos de la Pre-Entrega Implementados
- **Estructura Semántica HTML5:** Uso correcto de etiquetas principales como `header`, `nav`, `main`, `section` y `footer`.
- **Formulario de Contacto Activo:** Formulario funcional integrado con Formspree (solicitando Nombre, Email y Mensaje).
- **Diseño Responsivo** - Flexbox aplicado en la sección de tarjetas de productos (Tipos de Tour).
  - CSS Grid utilizado en la distribución estética de las reseñas.

- **Contenido Multimedia:** Inclusión de imágenes del club y un video/iframe del Estadio.
- **Tipografía:** Implementación de Google Fonts (Montserrat).
